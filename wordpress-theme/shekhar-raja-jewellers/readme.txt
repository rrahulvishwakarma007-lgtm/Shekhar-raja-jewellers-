A luxury jewellery WordPress theme for Shekhar Raja Jewellers.

== Description ==

Shekhar Raja Jewellers is a premium WordPress theme designed for luxury jewellery businesses. 
Built with elegance and sophistication in mind, this theme showcases gold, diamond, and bridal 
jewellery with stunning animations and a warm, luxurious design aesthetic.

= Features =

* **Responsive Design** - Looks great on all devices
* **Custom Post Type** - Dedicated Products section
* **Product Categories** - Organize jewellery by type
* **Hero Slider** - Animated homepage slider with Ken Burns effect
* **Gold Ticker** - Scrolling announcement bar
* **WhatsApp Integration** - Quick enquiry buttons throughout
* **Customizer Options** - Easy company info and gold rate updates
* **SEO Optimized** - Clean code and meta tags
* **Animation Ready** - Smooth CSS animations and 3D effects

== Installation ==

1. Upload the theme folder to `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Go to Appearance > Customize to configure theme settings
4. Set up your navigation menus in Appearance > Menus
5. Add your logo in Appearance > Customize > Site Identity

== Configuration ==

= Company Information =
Navigate to Appearance > Customize > Company Information to set:
* WhatsApp Number
* Established Year
* Company Tagline

= Gold Rates =
Navigate to Appearance > Customize > Gold Rates to update:
* 24K Gold Rate (other rates are auto-calculated)

= Products =
1. Go to Products > Add New
2. Add product title and description
3. Set featured image
4. Select product category
5. Add custom fields (optional):
   - product_tag: e.g., "Bestseller", "Premium", "Limited"
   - featured: set to "1" for featured products
   - price: e.g., "₹50,000 onwards"

= Categories =
Go to Products > Categories to manage product categories:
* Bridal
* Festive
* Diamond
* Rings
* Earrings
* Necklaces
* Bangles
