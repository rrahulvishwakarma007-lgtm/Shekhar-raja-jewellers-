=== Shekhar Raja Jewellers WordPress Theme ===

== Installation ==

1. Upload the 'shekhar-raja-jewellers' folder to /wp-content/themes/
2. Activate the theme via Appearance > Themes
3. Go to Appearance > Customize to configure settings

== Required Plugins ==

None required (works out of the box)

== Setup Guide ==

1. **Create Pages**
   - Home (use default template)
   - About (page-about.php template)
   - Contact (page-contact.php template)
   - Gold Rates (page-gold-rates.php template)
   - Bridal (page-bridal.php template)
   - App Download (page-app.php template)

2. **Configure Menus**
   - Go to Appearance > Menus
   - Create "Primary Menu" and assign to Primary location
   - Create "Footer Menu" and assign to Footer location

3. **Add Products**
   - Go to Products > Add New
   - Add title, description, and featured image
   - Select category and set featured/tag options

4. **Update Gold Rates**
   - Go to Gold Rates > Add New
   - Enter the 24K rate and date
   - Other rates are auto-calculated

5. **Customize Settings**
   - Go to Appearance > Customize
   - Update company info, phone, WhatsApp, address
   - Add social media links
   - Upload logo

== Shortcodes ==

[whatsapp_button text="Enquire" message="Hello!"]
- Displays a WhatsApp enquiry button

[gold_rates]
- Displays current gold rates

== Custom Post Types ==

- Products: For jewellery items
- Gold Rates: For daily gold rate updates
